## üìö Course Modules and Topics Covered

This course focused on mastering modern web layout techniques and responsive design using **HTML5** and **CSS3**.  
The main topics and modules covered include:

- **Advanced CSS Selectors:** Combining element, class, and attribute selectors for precise styling control.  
- **Interactive Pseudo-Classes:** Using `:hover`, `:focus`, and `:active` to enhance user interactions.  
- **Structural Pseudo-Classes:** Applying `:nth-child()`, `:first-child`, and `:last-child` for layout refinement.  
- **Flexbox Layout (Hobby Club Project):** Building vertically stacked and centered sections for responsive landing pages.  
- **Visual Effects & Transitions:** Implementing shadows, hover transitions, and color contrast for engaging UI design.  
- **CSS Grid Layout (Flavor Fiesta Project):** Creating complex, grid-based website structures with multiple sections.  
- **Responsive Design:** Adapting layouts for different screen sizes using flexible units and grid systems.
